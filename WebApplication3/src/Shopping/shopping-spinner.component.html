List Event and Spinner Area ///////////////////////

  • $ctrl.$docheck - send $rootScope.$broadcast('..', { on: true }) - loop promises.push(...promise call..) - loop final: $rootScope.$broadcast('..', { on: false })
  • As soon as $broadcast on, spinner.component.js monitors it using $rootScope.$on('..', function (event, data) { ...showSpinner=true... } and html uses ng-if="$ctrl.showSpinner" to show spinner
  • As soon as $broadcast off, spinner.component.js monitors it using $rootScope.$on('..', function (event, data) { ...stopSpinner=false... } and html uses ng-if="$ctrl.showSpinner" to hide spinner
